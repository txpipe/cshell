The `tx invoke` command allows you to invoke a [tx3](https://docs.txpipe.io/tx3) transaction (resolve, sign and submit).

## Usage

The arguments in this command are optional, if not provided, it runs in interactive mode.

```bash
cshell tx invoke [OPTIONS]
```

## Arguments

- `--tx3-file`: Path for Tx3 file describing transaction.
- `--tx3-args-json`: Json string containing args for the Tx3 transaction.
- `--tx3-args-file`: Path for file containing args for the Tx3 transaction.
- `--tx3-template`: Template for Tx3 file.
- `--signers`: Wallets that will sign the transaction.
- `--skip-submit`: Skip submitting.
- `--unsafe`: Allow signing with unsafe wallets.
- `--provider`: Name of the provider to use. If undefined, will use default.
