The `wallet restore` command allows you to restore a wallet using a BIP39 Mnemonic.

## Usage

```bash
cshell wallet restore [OPTIONS]
```

## Arguments

- `--name`: Name to identify the wallet.
- `--password`: Spending password used to encrypt the private keys.
- `--mnemonic`: BIP39 Mnemonic.
- `--unsafe`: Disable password requirement (not recommended).
