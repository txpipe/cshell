The `wallet balance` command shows the balance of a wallet.

## Usage

```bash
cshell wallet balance [OPTIONS]
```

## Arguments

- `name`: Name of the wallet to show the balance of. If undefined, will use default.
- `provider`: Name of the provider to use. If undefined, will use default.
- `--detail`: Whether to include details of all UTxOs or aggregated data.
