The `wallet edit` command allows you to edit an existing wallet.

## Usage

```bash
cshell wallet edit [OPTIONS]
```

## Arguments

- `name`: Name of the wallet to update. If undefined will use default.
- `--new-name`: New name for wallet.
- `--is-default`: Whether to set as default wallet.
