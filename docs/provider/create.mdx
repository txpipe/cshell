The `provider create` command allows you to create a new provider service. This command requires a UtxoRPC and TRP provider, the UtxoRPC is to fetch on-chain data, and the TRP is to resolve and submit transactions.

<Aside type="tip">
Use [demeter](https://demeter.run/) get free UtxoRPC and TRP instance.
</Aside>

## Usage

The arguments in this command are optional, if not provided, it runs in interactive mode.

```bash
cshell provider create [OPTIONS]
```

## Arguments

- `--name`: Name to identify the provider.
- `--is-default`: Whether to set as default provider.
- `--network-kind`: Whether it is mainnet or testnet.
- `--utxorpc-url`: UTxORPC url
- `--utxorpc-headers`: key:value UTxORPC headers
- `--trp-url`: TRP url
- `--trp-headers`: key:value TRP headers
