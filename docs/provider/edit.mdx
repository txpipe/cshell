The `provider edit` command allows you to edit an existing provider service.

## Usage

```bash
cshell provider edit [OPTIONS]
```

## Arguments

- `name`: Name to identify the provider.
- `--new-name`: New name to identify the provider.
- `--is-default`: Whether to set as default provider.
- `--network-kind`: Whether it is mainnet or testnet.
